# C-Tab 新标签页

C-Tab 新标签页是一款免费浏览器扩展，采用简洁优雅的设计，为您打造更高效、更个性化的浏览体验。

## 功能特性

- **快捷方式管理** - 支持普通标签和图文标签两种样式
- **分类管理** - 自定义分类，拖拽排序，设置专属颜色
- **主题切换** - 支持深色/浅色两种主题
- **搜索功能** - 集成搜索栏，快速搜索
- **数据备份** - 支持导入导出，数据不丢失
- **批量操作** - 支持批量删除、移动快捷方式

## 安装方法

### Edge 商店安装（推荐）

1. 访问 Microsoft Edge Add-ons 商店
2. 搜索 "C-Tab 新标签页"
3. 点击"获取"按钮安装

### 手动安装

1. 下载最新版本的扩展包
2. 解压到本地目录
3. 打开 Edge 浏览器，访问 `edge://extensions/`
4. 开启"开发人员模式"
5. 点击"加载解压缩的扩展"
6. 选择解压后的目录
